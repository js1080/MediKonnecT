영상과 함께 이해에 도움이 되도록 간단하게 각 기능들에 대해 정리해보았습니다.



환자:

1. 의사와 상담할 수 있는 1:1 채팅방 기능
2. '오늘의 복약' : 오늘의 복약 상태를 체크함. 복약 후 부작용 발생 시 위급 알람을 의사에게 보낼 수 있음.
3. 진료 예약: 온라인으로 시간, 날짜 선택 후 예약 가능
4. Q&A: 질문 커뮤니티 창, 모든 유저의 과거질문과 의사의 답변을 확인할 수 있음.
5. 진료기록부: 의료진이 업로드한 과거 자신의 진료기록부를 다운받고 확인할 수 있음.


보호자:

1. 가족관계 인증: 가입돼있는 환자중 자신의 피보호자를 찾고, 가족 관계증명서를 업로드시 인증받을 수 있음. (이미 인증된 환자는 뜨지 않음.)
가족 관계 증명 이전에는 환자의 정보를 열람할 수 없음.
2. 복약 모니터링: 환자가 복약을 잘 했는지 복약률을 확인할 수 있고, 부작용 발생 알람 시 보호자에게도 알림이 옴.
3. 대리 예약: 피보호자 중 선택하여 원하는 날짜와 시간에 진료 예약을 대리로 가능함.
4. 진료기록부: 피보호자의 진료기록부를 공유할 수 있음.
5.  Q&A: 질문 커뮤니티 창, 모든 유저의 과거질문과 의사의 답변을 확인할 수 있음.


의료진: 

1. 환자상담: 담당환자 중 1명을 택해 상담 채팅 가능
2. 환자 관리 및 처방: 담당 환자 목록이 뜨고, 약 처방(실제 약처방은 오프라인으로, 복약 확인용), 진료기록부 업로드, 관리 해제 등 기능이 있음.
   '새 환자 추가하기'로 진료받은 환자를 추가할 수 있음 (단, 환자가 이 서비스에 가입해야함)
3. 복약지도 모니터링: 담당환자들의 복약률을 확인할 수 있음, 환자가 부작용 발생 시 위급알림이 오고 복약지도를 통해 환자가 어떤 부작용을 앓고 있는지 확인 가능함. (환자가 타이핑한 대로)
4. 일정관리: 일주일 간격으로 현재 진료 예약 전체 상태를 확인할 수 있음.
5. Q&A: 환자나 보호자가 올린 질문에 답변할 수 있음.
